# LOCMACHINE
TRABALHO DE CONCLUSÃO DE CURSO. Proposta da empresa LOCMACHINE.
A LocMachine é uma plataforma desenvolvida para facilitar a conexão entre consumidores e prestadores de serviços que trabalham com máquinas pesadas, como tratores, guindastes, escavadeiras e muito mais.
O objetivo do projeto é centralizar informações, agilizar processos de locação e tornar o acesso a equipamentos mais eficiente, tanto para quem precisa contratar quanto para quem oferece.

O site oferece uma interface simples, intuitiva e responsiva, permitindo ao usuário:

Explorar máquinas disponíveis para locação

Visualizar detalhes como preço, categoria e disponibilidade

Entrar em contato com prestadores

Navegar facilmente entre categorias e páginas

Acessar a plataforma tanto no computador quanto em dispositivos móveis

Este projeto foi desenvolvido como parte de uma demanda de um TCC visando propor uma solução para um mercado real.
